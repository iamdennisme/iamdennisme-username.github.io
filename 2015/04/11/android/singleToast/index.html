Posted on 2015-04-11 | In Android , AndroidUtil

一个自己写的工具类，用来Toast的，避免重复Toast。

大家肯定都用过，但是如果你写一个button，button的click就是这段代码，如果你连续点击20次button，我估计到你关掉应用，这个Toast还在不停的Toast。

所以有了这个工具类。

#完整代码
git:ToastUtil

#方法
show(Context context, String message, int showTime)

这个方法其实是只做一件事，就是指定Toast的时间，指定后调用show(Context context, String message)，有点需要注意的是：

在Toast源码中，有这么一段

通过@IntDef注解限定了Toast的时长只能是SHORT或者LONG，两个定义好的， 所以在这里检测了show的时长输入，如果有问题，直接拋异常。

show(Context context, String message)

主要的实现基本都在这个类中，判断Toast是否为null，如果为null就创建新的，同时，记录上一次的toast信息和时间，如果Toast信息是一样的，则计算两次toast间隔时间，如果不到toast时常，则不进行toast，当然，如果是不同的信息，则Toast，以免消息漏掉。
