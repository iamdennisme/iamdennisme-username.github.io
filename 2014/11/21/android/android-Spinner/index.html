Spinner

Posted on 2014-11-21 | In Android , AndroidDemo

#Spinner

提供一个下拉选单，用于在一堆数据列表中选择一个值，未展开显示当前选中的值，点击后出现选单，展示所有可选择的值，可选择新值。

#简单实现

##xml

一些属性：

  • android:spinnerMode=”dialog”:
    这个属性定义了spinner的展现方式，有两种dialog和dropdown。分别是弹窗和下拉

    dialog

    dropdown

  • android:entries其实是定义了spinner菜单里的数据，这里指向了array。

但其实在java代码中也可以添加数据。先简单实现

  • android:prompt="@string/title"如果是dialog模式，显示tiitle

##java

简单使用的话，就一个setOnItemSelectedListener

可以看到两个回调onItemSelected和onNothingSelected

  • 关于onItemSelected
    这个方法是在你改变了spinner的选择值时触发。两个注意点：

    • 默认值也会触发这个回调。在spinner实现完成，就算没有主动去选择值，但会默认显示一个，这个时候也会触发回调.
    • 如果你对spinner进行了操作，但是并没有改变当前spinner的值，那是不会触发这个回调的。
  • 关于onNothingSelected，会在后面说到

#Spinner的Adapter
上面我们说到，其实可以不用xml来绑定spinner的数据，那就需要adapter来绑定数据

##实现

是不是感觉好面熟，这个套路，对，简直和listview一模一样对不对。

but，你肯定发现了。setDropDownViewResource是什么，好吧
官方解释

Sets the layout resource to create the drop down views.
因为adapter的第二个参数设置的样式是其没有展开时的样式，而这里是设置下拉展开的样式。

对，你是不是发现了什么，我们可以通过setDropDownViewResource来自定义spinner的样式。举个栗子，如果你不设置样式。样式是这样的。
